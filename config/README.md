# Directory: config

This directory contains configuration files for the Automated ICD Diagnosis Extraction Project.

## Files

- `config.json` - Main configuration file containing:
  - Spark configuration settings
  - NLP processing parameters
  - Data input/output paths
 - Project metadata

## Usage

The configuration file is loaded by the main application to set up the processing environment. You can modify these settings to adjust the behavior of the ICD diagnosis extraction pipeline.
