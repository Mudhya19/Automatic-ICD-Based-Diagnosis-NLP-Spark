# Utils Directory

This directory contains utility components and helper functions used in the dashboard application.

## Contents
- Helper functions for data processing
- Utility modules for the dashboard components
- Common functions shared across the application
- Configuration utilities

## Purpose
The utilities in this directory provide modular functionality that supports the main dashboard application. These components help in:
- Data preprocessing and cleaning
- Model loading and prediction functions
- Visualization helper functions
- Common operations used across different parts of the application

## Components
The utils directory includes:
- Data processing functions
- Model interaction utilities
- Dashboard component helpers
- Error handling and logging utilities
- Configuration management functions
