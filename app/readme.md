# App Directory

This directory contains the main dashboard application for the Automated ICD Diagnosis Extraction project.

## Contents
- `dashboard.py`: Main Streamlit dashboard application
- Configuration files for the dashboard
- Supporting assets for the web interface

## Dashboard Features
The dashboard provides an intuitive web interface for:
- Uploading medical records for processing
- Visualizing ICD diagnosis extraction results
- Interactive exploration of medical data
- Model performance metrics and insights

## Usage
To run the dashboard application:
1. Ensure all dependencies are installed
2. Activate the virtual environment
3. Run `streamlit run app/dashboard.py`

The dashboard will be accessible through your web browser at the provided URL.

## Components
The application is built with Streamlit and includes:
- File upload functionality for medical records
- Real-time processing and visualization
- Interactive charts and tables
- Model prediction outputs
